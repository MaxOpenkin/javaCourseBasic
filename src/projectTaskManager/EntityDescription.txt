
1. Project (Проект).
Описание: Проект представляет собой основную единицу, вокруг которой строится вся работа.
Он содержит информацию о названии проекта, его описании, сроках выполнения, участниках и задачах,
связанных с данным проектом.
Атрибуты:
- ID: Уникальный идентификатор проекта.
- Название: Название проекта.
- Описание: Краткое описание проекта и его целей.
- Дата начала: Планируемая дата начала работы над проектом.
- Дата окончания: Планируемая дата завершения проекта.
- Список задач: Ссылки на задачи, входящие в состав проекта.
- Участники: Список пользователей, участвующих в проекте.

2. Task (Задача).
Описание: Задача — это конкретное действие или набор действий, необходимых для достижения целей проекта.
Задачи назначаются на конкретных участников проекта и имеют сроки выполнения, приоритет и статус выполнения.
Атрибуты:
- ID: Уникальный идентификатор задачи.
- Название: Название задачи.
- Описание: Детальное описание задачи.
- Приоритет: Приоритет задачи, определяющий важность выполнения.
- Статус: Текущий статус выполнения задачи (например, "В работе", "Выполнена").
- Срок выполнения: Дата, к которой задача должна быть выполнена.
- Комментарии: Список комментариев к задаче.
- Назначенный пользователь: Пользователь, на которого назначена задача.

3. User (Пользователь).
Описание: Пользователь — это любой человек, использующий систему.
Пользователи могут иметь разные роли в проектах (например, менеджер проекта, исполнитель задач).
Атрибуты:
- ID: Уникальный идентификатор пользователя.
- Имя пользователя: Имя, используемое пользователем в системе.
- Email: Адрес электронной почты пользователя.
- Пароль: Хешированный пароль пользователя.
- Роли в проектах: Список ролей пользователя в различных проектах.

4. Comment (Комментарий).
Описание: Комментарии используются для обсуждения задач между участниками проекта.
Каждый комментарий связан с конкретной задачей.
Атрибуты:
- ID: Уникальный идентификатор комментария.
- Автор: Пользователь, создавший комментарий.
- Текст: Содержание комментария.
- Дата создания: Время создания комментария.
- Привязка к задаче: Ссылка на задачу, к которой относится комментарий.
