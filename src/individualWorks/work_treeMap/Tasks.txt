
TreeMap отлично подходит для задач,
где важен порядок элементов или необходимо быстро выполнять операции поиска,
вставки и удаления с учетом ключей.

1. Управление записями сотрудников
Описание: Создайте систему для управления записями сотрудников в компании.
Каждый сотрудник имеет уникальный идентификатор (ID), имя, должность и зарплату.
Система должна позволять добавлять, удалять и искать записи сотрудников по их ID.
Также реализуйте функцию для вывода всех записей сотрудников, отсортированных по их ID.

2. Система рейтинга игроков
Описание: Разработайте систему рейтинга для игроков онлайн-игры,
где каждый игрок имеет имя и рейтинговые очки.
Используйте TreeMap для хранения записей игроков, где ключом будет рейтинговое число,
а значением — имя игрока. Система должна поддерживать добавление новых игроков,
обновление рейтинга существующих игроков и вывод списка игроков по рейтингу.

3. Индекс слов в тексте
Описание: Напишите программу, которая анализирует текст и создает индекс всех слов,
встречающихся в тексте, где каждому слову соответствует список номеров страниц,
на которых это слово встречается.
Используйте TreeMap, где ключом будет слово, а значением — список номеров страниц.
Это позволит быстро находить, на каких страницах встречается каждое слово.

4. Управление заказами в ресторане
Описание: Создайте систему для управления заказами в ресторане,
где каждый заказ имеет уникальный номер,
статус (например, "принят", "готовится", "готов") и время создания заказа.
Система должна позволять добавлять новые заказы,
изменять их статус и удалять заказы из системы.
Используйте TreeMap для сортировки заказов по времени создания.

5. Система бронирования номеров в отеле
Описание: Разработайте систему для управления бронированием номеров в отеле.
Каждое бронирование имеет уникальный номер, дату заезда и выезда, и имя гостя.
Система должна позволять добавлять и удалять бронирования,
а также искать бронирования по дате заезда.
Используйте TreeMap для организации бронирований по дате заезда.
