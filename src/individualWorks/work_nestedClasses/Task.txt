
Практические задачи на тему вложенных классов

Задача 1: Система управления задачами
Разработайте простую систему управления задачами для команды разработчиков.
Используйте внутренний класс для представления задачи, которая содержит информацию,
такую как описание задачи, приоритет и статус выполнения. Внешний класс должен управлять списком задач,
позволяя добавлять новые задачи и выводить список задач с определенным статусом.

Задача 2: Система бронирования отелей
Создайте систему бронирования для сети отелей.
Используйте статический вложенный класс для представления информации о бронировании,
включая номер бронирования, даты заезда и выезда. Внешний класс должен представлять отель,
содержащий список доступных номеров и методы для выполнения бронирования.

Задача 3: Финансовый калькулятор
Реализуйте финансовый калькулятор для расчета ежемесячных платежей по кредиту.
Используйте анонимный класс для реализации интерфейса с методом расчета.
Внешний класс будет содержать методы для ввода суммы кредита, процентной ставки и срока.

Задача 4: Логирование действий пользователя
Создайте систему логирования, которая позволит записывать действия пользователя в приложении.
Используйте локальный класс внутри метода для генерации уникального идентификатора лога
и записи информации о действии пользователя. Внешний класс должен управлять логами,
позволяя добавлять новые записи и выводить их.

Задача 5: Менеджер настроек приложения
Разработайте систему управления настройками приложения,
где настройки могут быть различными: графическими, системными, пользовательскими и т.д.
Используйте внутренние классы для разделения настроек по категориям,
а внешний класс для управления и сохранения всех настроек.

Задача 6: Модуль создания отчетов
Создайте модуль для генерации отчетов по продажам в компании.
Используйте анонимные классы для определения различных форматов отчетов (например, PDF, HTML, текстовый формат).
Внешний класс должен позволять пользователю выбрать формат и сгенерировать отчет с данными о продажах.

Эти задачи помогут вам практически применить знания о вложенных классах в Java
и разработать решения для конкретных бизнес-сценариев и проектных задач.
Попробуйте реализовать каждую из них, чтобы укрепить свое понимание вложенных классов
и их применения в реальных условиях.
